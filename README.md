# To-Do-list-app
A To-Do List Manager which has acceptable design and features built with Python Tkinter

## Features
- Add, edit, and delete tasks
- Priority levels (High, Medium, Low)
- Filter tasks by priority or status
- Color-coded tasks based on priority
- Data persistence with JSON storage
- Keyboard shortcuts
- Task completion tracking with statistics

## Installation

```bash
# Clone the repository
git clone https://github.com/YOUR_USERNAME/todo-list-app.git

# Navigate to the directory
cd todo-list-app

# Run the application
python todo_app.py
```
